# 💰 Expense Management App

A full-stack Expense Management Application that helps users track, manage, and analyze their daily expenses efficiently with a clean and responsive user interface.

---

## 🚀 Features

- 🔐 User Authentication
- ➕ Add Expenses & Income
- ✏️ Edit/Delete Transactions
- 📊 Expense Analytics Dashboard
- 📂 Category-wise Expense Tracking
- 📅 Date-based Filtering
- 📱 Fully Responsive Design
- ⚡ Real-Time Updates

---

## 🛠️ Tech Stack

### Frontend

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/harish-joya/Expense-Management-App.git
```

### 2️⃣ Move to Project Folder

```bash
cd Expense-Management-App
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Run the Project

```bash
npm run dev
```
